Java--

Meine eigene Programmiersprache

Idee

Projektfindung

Kurz nach dem Schülerstudium suchte ich eine neue Herrausforderung, wo ich mein Wissen umsetzen konnte. Ich guckte mir viele Themen an und bin schließlich bei künstliche Intelligenz und Compilerbau gelandet. Bei beiden Themen gibt es viel Wissen kostenfrei und einfach zur Verfügung. Am Ende entschied ich mich für den Compilerbau, weil ich schon immer wissen wollte, wie das genau unter der Haube funktioniert.

Der Name

Schnell stand auch fest, wie ich meine "eigene Programmiersprache" nennen wollte. Java++. "Java", weil ich die Sprache am Java-Syntax (und Python) anlehnen wollte und dass "++" hat den gleichen Grund wie bie C++ (klick hier). Später wurde mir erst bewusst, wie viele Funktionen so eine Programmiersprache eigentlich hat und dass ich so viel (zeitlich) erstmal nicht bieten kann und benannte sie dann zu "Java--" um.

Umsetzung

Zu erst verschaffte ich mir einen Überblick über den Ablauf des Kompilierens. Weiter beschloss ich, dass man erstmal nur Code in das Programm reingeben kann und diesen Ausführen kann. Das Kompilieren in eine andere Sprache ist erstmal nicht vorgesehen. Danach begann ich mit ersten zwei Phasen, das Erfassen und Aufbereiten des Quellcodes. Features

Variablen

Variablen gehören zur fast jeder Programmiersprache dazu und sind unverzeichtbar. Die Basisdatentypen sind: char, string, int, long, float, double und boolean. Es ist nicht erlaubt Zahlen im Variablename zu benutzen. Die Variablen, die mit einen "$"-Symbol beginnen sind von der Programmiersprache vorgegeben.

Deklariern

Syntax: [Datentyp] [Name] = [Wert]

Beispiele:

int var = 42
string varTwo = "Hello World"
double varThree = 842.75

Zuweisen

Syntax: [Name] = [Wert]

Beispiele:

var = 40 + 2
var = var + 1
string varTwo = "Hello" + " World"
double varThree = 800 + 42.75

Funktionen

Auch Funktionen sind essentiell. Man kann sich wiederholdenden Code abspeichern und immer wieder aufrufen. Sie können auch einen Wert zurückgeben.

Definieren

Syntax:

def [Rückgabetyp] [Name] ( [[Attribut 1] , [Attribut 2] , ...] ) {
     [Code...]
}

Beispiele:

def int add ( int x , int y ) {
     int result = x + y
     return result
}
def void sayHello ( string name ) {
     print ( "Hello, " )
     println ( name )
}

Aufrufen

Syntax: [Name] ( [[Parameter 1] , [Parameter 2] ...] )

Ein paar Funktionen sind von der Programmiersprache vorgegeben.

Name Attribut(e) Rückgabetyp Funktion Beispiel(e)
print object message void Schreibt den Wert des Parameters in der Konsole ohne \n print ( "Hello World" )
println object message void Schreibt den Wert des Parameters in der Konsole mit \n am Ende println ( "Hello World" )
dump string type void Zeigt Informationen über Variablen oder Funktionen. Als Parameter: "variables" - für Variablen, "functions" - für Funktionen
dump ( "variables" )
dump ( "functions" )
input string message string Sendet eine Nachricht und fordert den Benutzer auf eine Eingabe zu tätigen
input ( "Was ist dein Alter?: " )
string alter = $USER_INPUT
clearConsole Keine void Leert die Konsole clearConsole ( )
exit Keine void Beendet das Programm exit ( )
