Tell The World

Kurznachrichtendienst, um seine Gedanken zu teilen

Die Idee

Projektfindung

Ich wollte mal ein komplettes großes Projekt umsetzen. Weil ich nicht gut darin bin, selber neue Projektideen zu finden, habe ich gegooglet. Ich hab einen guten Tipp gefunden, wobei es darum geht eine vorhandene Website "nachzumachen" oder sich an dieser zu inspirieren. Nach kurzer Überlegung habe ich mich entschieden eine Art Twitter-Klon zu machen.

Der Name

Den Name TellTheWorld habe ich davon abgeleitet, was man mit der Website machen kann - etwas der Welt erzählen. Auch wenn es nur ein "Lernprojekt" ist, habe ich später noch die Möglichkeit die Website zu veröffentlichen.

RESTful API

Ungefär zur gleichen Zeit, wo ich das Projekt gestartet habe, habe ich auch RESTful entdeckt und wollte dies ebenfalls in diesen Projekt mit einbinden. Da ich ebenfalls mal etwas mit Python machen wollte, habe ich die RESTful API mit flask und flask_restful umgesetzt

Den Code zum Python-Backend findest du hier: Klick hier

API Benutzen

Repo und notwendige Pakete herunterladen

Mit Python starten: python3 ./Code/Backend/main.py

Informationen über einen User: localhost:8080/api/v1/user/(user_tag)/

Informationen über eine Nachricht: localhost:8080/api/v1/message/(message_id)/
