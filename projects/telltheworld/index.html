Tell The World

Kurznachrichtendienst, um seine Gedanken zu teilen

Beschreibung

Projektfindung

Ich wollte mal ein komplettes großes Projekt umsetzen. Weil ich nicht gut darin bin, selber neue Projektideen zu finden, habe ich gegooglet. Ich hab einen guten Tipp gefunden, wobei es darum geht eine vorhandene Website "nachzumachen" oder sich an dieser zu inspirieren. Nach kurzer Überlegung habe ich mich entschieden eine Art Twitter-Klon zu machen.

Der Name

Den Name TellTheWorld habe ich davon abgeleitet, was man mit der Website machen kann - etwas der Welt erzählen. Auch wenn es nur ein "Lernprojekt" ist, habe ich später noch die Möglichkeit die Website zu veröffentlichen.

Features

Das Hauptfeature ist, dass man Kurznachrichten veröffentlichen kann. Andere können diese Nachricht Liken, Kommentieren oder Reposten. Außerdem kann man andere Benutzer folgen, sodass man die neusten Nachrichten von der Person ebenfalls auf seiner Startseite erscheinen. Auch soll man die Möglichkeit haben andere Benutzer privat über einen Chat zu kontaktieren und sich darüber auszutauschen.

RESTful API

Über eine RESTful API soll man die Möglichkeit haben, verschiedene Informationen und Statistiken über Benutzer, Nachrichten und der Plattform zu bekommen. Implementiert wird diese in Java mit Spring Boot.
